思维挑战8：哥德巴赫猜想- if 语句
-

[代码资源](https://download.csdn.net/download/Btbsja/12155114)

现代版本的哥德巴赫的猜想为以下两种：
1.任一大于5的整数都可以写成3个质数之和；
2.任一大于2的偶数都可以写成2个质数之和。

接下来我们验证一下100以内大于2 的偶数都可以写成2个质数之和。

    //C032
    #include <stdio.h>
    #include <stdlib.h>
    int main()
    {
        int k,a,b,i,count1,count2;
        for(k=4;k<=100;k=k+2)
        {
            for(a=2;a<=k/2;a++)
            {
                count1=0;
                for(i=2;i<=a-1;i++)
                {
                    if(a%i==0)
                    {
                        count1++;
                        break;
                    }
                }
                if(count1==0)
                {
                    b=k-a;
                    count2=0;
                    for(i=2;i<=b-1;i++)
                    {
                        if(b%i==0)
                        {
                            count2++;
                            break;
                        }
                    }
                    if(count2==0)
                    {
                        printf("%d=%d+%d\n",k,a,b);
                        break;
                    }
                }
            }
        } 
        system("pause");
        return 0;
    }
