# 网络监视和流量分析

4.1 实验目的

**1. 理解什么是网络证据，应该采取什么办法收集网络证据。**

**2. 了解网络监视和跟踪的目的，会用 WinDump 进行网络监视和跟踪。**

**3. 使用 Ethereal 软件分析数据包，查看二进制捕获文件，找出有效的证据。**

4.2 实验环境和设备

**1. Windows XP 或 Windows 2000 Professional操作系统；**

**2. 网络运行良好；**

**3. WinPcap、WinDump 和 Ethereal 安装软件。**
